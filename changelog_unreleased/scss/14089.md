#### Fix removing commas from function arguments in maps (#14089 by @sosukesuzuki)

<!-- prettier-ignore -->
```scss
/* Input */
$foo: map-fn(
  (
    "#{prop}": inner-fn($first, $second),
  )
);

/* Prettier stable */
$foo: map-fn(("#{prop}": inner-fn($first $second)));

/* Prettier main */
$foo: map-fn(
  (
    "#{prop}": inner-fn($first, $second),
  )
);

```
